# Rick and Morty GraphQL App

This project is part of the ALX GraphQL module (0x01).  
It demonstrates how to set up a Next.js application using TypeScript, TailwindCSS, Apollo Client, and GraphQL.  
The app connects to the Rick and Morty GraphQL API to fetch episode data.
